Early Years

The Computer

Digital technology is used to store and process data into useful information. Humans have been inventing tools to solve problems throughout the history and a machine to solve thinking problems is no exception.

Early computer scientists realized a thinking machine have to perform certain tasks, like take input, store information, process information and output results.

These four different tasks are common to all computers, and what makes a computer, a computer!

Charles Babbage

Charles Babbage is called the father of the computer and was a mathematician and mechanical engineer who designed the Analytical Engine. It's considered to be the first ever general-purpose computer, even though it was never built.

The Analytical Engine was first described in 1837 and was a mechanical object, meaning it would be powered by steam or some similar source of power (this was before vacuum tubes and transitors). It used a complex mechanism which allowed the machine to repeat a deck of punch cards, so as to execute a loop.

Ada Lovelace

Ada Lovelace or Ada King, countess of Lovelace is often referred to as the first programmer because she developed the first algorithm for the Analytical Enginge. While Babbage designed the machine, Lovelace actually described what you could do with it.

Fundamentally computers are for executing algorithms, which is basically a series of steps that solves a specific problem. Lovelace described the first computer algorithm that could compute Bernoulli numbers. Since the machine was never completed, her algorithm was never tested.

The 1900s

Alan Turing

Alan Turing was a mathematician, computer scientist, cryptanalyst and philosopher and is considered to be the father of theoretical computer science and artificial intelligence. He developed the Turing machine, which can be considered a model of a general-purpose computer and Turing played a key part in cracking the german Enigma Machine during World War II.

A Turing complete system means a system in which a program can be written that will find an answer (although with no guarantees regarding runtime or memory), in other words, a device that can solve any computational problem.

The Turing test, originally called the imitation game by Alan Turing is a deceptively simple method of determining whether a machine can demonstrate human intelligence. If a machine can engage in a conversation with a human without being detected as a machine, it has demonstrated human intelligence.

The ENIAC

The Electronic Numerical Integrator and Computer was the first electronic general-purpose digital computer. It was Turing complete, and able to solve problems through reprogramming and had around 17000 vacuum tubes.

Vacuum tubes work by removing the air from a tube to create a vacuum which gives the electrons a clear path from the cathode to the anode, which creates a current.

Vacuum tubes were replace by transitors to control electricity voltages. Transistors could replicate all the functions of tubes, like switching and amplification, but was made out of semiconductor materials which made them more compact and more efficient.

Grace Hopper

Admiral Grace Hopper was an American computer scientist and United States Navy rear admiral. She is known for inventing the very first compiler, A-0 and developed Flow-Matic, the first programming language to use English-like commands.

Compilers made it possible to translate human language via a programming language into machine code. That way programmers didn't need to learn how to write machine code in ones and zeros.

Modern Era

The Xerox Alto

The development of hard disk drives and silicon microprocessors made computers smaller. Instead of filling up entire rooms like the ENIAC, they were small enough to fit on tabletops.

The Xerox Alto was introduced on March 1, 1973 and was the first computer that resemble computers we're familiar with today. It was also the first computer to implement a graphical user interface, or GUI that used icons, a mouse, and a window.

Personal Computer

In the 1970s, a young engineer named Steve Wozniak invented the Apple I, a single-board computer for hobbyists. With his friend Steve Jobs, they created the company Apple Computer. The Apple II and later the Apple Macintosh with MacOS using a graphical user interface (GUI) were phenomenal successes, giving a new generation of people access to personal computers.

In the 1980s, IBM introduced its personal computer with a primitive version of an operating system called MS-DOS or Microsoft Disk Operating System, thanks to a partnership with Microsoft and Bill Gates. In 1985 Windows was introduced as a graphical operating system shell for MS-DOS in response to the Apple Macintosh realeased the year before.

Open Source

Unix is an operating system developed by Ken Thompson and Dennis Ritchie at AT&T Bell Labs. Since Unix wasn't cheap or accessible to the public, a programmer by the name of Richard Stallman started developing the GNU project in 1983. It was a free to use operating system with similar functionality to Unix.

Unlike Windows or MacOS, GNU wasn't owned by a single company, its code was open source, which meant that anyone could modify and share it.

By 1990 the GNU project had all the major components except the kernel. In 1991 Linus Torvalds released a Unix-like kernel called Linux. Combined with the tools developed by the GNU Project, Linux became one of the largest open source projects in history.
